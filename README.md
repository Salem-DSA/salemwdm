# salemwdm 

The platform site for Salem Workers Deserve More. Salem Workers Deserve More is the priority campaign of the [Salem Chapter of the Democratic Socialists of America](https://www.salemdsa.org/).

## Built With

* [Node (24.12.0)](https://nodejs.org/en/download)
* [React (19.2.0)](https://react.dev/)
* [Vite (7.2.4)](https://vite.dev/)
* [Bootstrap (5.3.8)](https://www.npmjs.com/package/bootstrap)

## Build Commands

Compile and run:
```
npm run dev
```

Compile and build to dist/:
```
npm run build
```

Preview local build:
```
npm run preview
```

Deploy dist/ to gh-pages:
```
npm run deploy
```
